What's included

An agent with superpowers, ready in minutes

Memory, automation, communications, and the ability to create its own skills. All inside the agent you already use.

Persistent memory

It remembers decisions, findings and agreements across sessions (Engram, on SQLite). Stop re-explaining the context to your agent every time.

46+ skills on demand

Skills that load only when needed — fast and without cluttering the context. From development to security.

WhatsApp auto-reply

It works with a regular number, no business account required. To strangers it replies as your "persona," with your style and your rules.

Home automation

Control Philips Hue, Home Assistant and devices over MQTT directly from your agent.

It extends itself

Need a new skill? Describe it and Dexter creates and activates it instantly, no restart required.

Desktop automation + 112 ready-made flows

Describe a task in natural language and Dexter takes screenshots, reasons about the screen, and runs clicks and keystrokes until it's done — saving the sequence as a reusable macro. Plus over 112 community flows (ClawFlows) ready to import.

How it works

One command and you're done

The installer detects your agent and does the rest — backing up your configuration first.

Install

A script (install.sh) that backs up your current config.

Detect

It recognizes your agent: Claude Code, Cursor, Codex, OpenCode…

Inject

It adds the system prompt and copies the 46+ skills.

Empower

Your agent boots up with memory, automation and more.

Under the hood

Technology

Multi-agent

Claude Code, OpenCode, Codex, Cursor, Gemini CLI and VS Code — the same ecosystem across all of them.

Local memory (Engram)

SQLite + full-text search. It stores and retrieves context across sessions, on your machine.

Open source

Open source under Apache 2.0, in active development and with automated tests.
